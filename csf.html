Description:

I currently tutor VCE mathematics, and wanted to create an app which allowed my students to track their study. I used the Django web development framework to create this. This was my first complete project with Django, and I was learning by doing - I taught myself how to interact with data models, work with an MVC architecture and design a solid user experience.

Technologies used:

  • Django (a web framework built on Python)
  • Bootstrap (grid system and styling)
  • Github (version control)
  • PythonAnywhere (host and run Python)
  • ChartJS (simple javascript graphs)

Deploying and hosting online

So that my students could use the site, I purchased the domain coolstudyfriend.com. I bought some basic web hosting to host a static marketing site. I installed the Django app on the subdomain app.coolstudyfriend.com, hosted and deployed through PythonAnywhere.

Product management (interaction with users):

CoolStudyFriend was my first project with live users. This exposed me to requested features, fixing bugs and prioritising tasks. I created a Trello board which allowed me to track and organise the tasks at hand.

Creating an API

To challenge myself with this project, I built a small API which allowed for easy interactions with my student's data. I used this API to implement ChartJS - through this plugin, I could easily create graphs which showed the hours students had dedicated to study throughout the week.
